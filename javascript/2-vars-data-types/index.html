Variables and Data Types

Topics

In JavaScript, variables are used to store data that can be referenced and manipulated throughout the program. Here’s an overview of how to declare variables and the available data types:

1. Declaring Variables

Variables can be declared using the following keywords:

  • var: Declares a variable, can be re-declared and updated.
  • let: Declares a variable, can be updated but not re-declared within the same scope.
  • const: Declares a constant, which cannot be updated or re-declared.

Example of Declaring Variables:

                
                    var name = "John"; // Can be reassigned
                    let age = 30;      // Can be reassigned but not redeclared
                    const country = "USA"; // Cannot be reassigned
2. JavaScript Data Types

JavaScript has 7 primitive data types:

  • String: A sequence of characters (text).
  • Number: Numeric values (integers or floating-point).
  • Boolean: A true or false value.
  • Undefined: A variable that has been declared but not assigned a value.
  • Null: A variable explicitly set to have no value.
  • Symbol: A unique identifier, introduced in ES6.
  • BigInt: Represents large integers, introduced in ES11.

Example of Variables with Different Data Types: 

                
                    let message = "Hello, world!"; // String
                    let count = 100;               // Number
                    let isActive = true;           // Boolean
                    let unknown = undefined;       // Undefined
                    let emptyValue = null;         // Null
                    let uniqueID = Symbol("id");   // Symbol
                    let largeNumber = BigInt(123456789012345678901234567890); // BigInt
