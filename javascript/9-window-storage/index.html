Window Objects, Methods, and APIs

Topics

Window Object

The window object in JavaScript is the global object that represents the browser's window or tab. It provides access to the browser’s features, including the DOM, the console, and other properties and methods for interacting with the browser's environment. It is the root object in the browser, and almost everything in JavaScript is a property of the window object.

Global Object

In the browser environment, the window object serves as the global object. This means that all global variables, functions, and objects are properties of the window object. 

var x = 10; console.log(window.x); // 10
function greet() { console.log("Hello!"); } window.greet(); // "Hello!"
Browser Information
  • navigator: Provides information about the browser and operating system. 
    • // Example
console.log(navigator.userAgent); // Outputs the browser's user agent string
  • location: Contains the current URL and allows navigation.
    • // Example
console.log(location.href); // Logs the current URL
location.reload(); // Reloads the page
  • history: Lets you navigate through the browser's session history.
    • // Example
history.back(); // Goes back one page
history.forward(); // Goes forward one page
Window Dimensions
  • window.screen: Provides information about the screen (width, height, etc.).
  • innerWidth and innerHeight: Dimensions of the content area. 
    • // Example
console.log(window.innerWidth, window.innerHeight); // Logs the width and height of the content area
  • outerWidth and outerHeight: Total dimensions including toolbars.
    • // Example
console.log(window.outerWidth, window.outerHeight); // Logs the total width and height
Scrolling
  • scrollX and scrollY: Get the current scroll positions.
    • // Example
console.log(window.scrollX, window.scrollY); // Logs the horizontal and vertical scroll positions
  • scrollTo: Scrolls to a specific position.
    • // Example
window.scrollTo(0, 500); // Scrolls to 500px from the top
Timers
  • setTimeout: Executes a function after a specified delay.
    • // Example
setTimeout(() => {
    console.log("Timeout executed!");
}, 2000); // Executes after 2 seconds
  • setInterval: Repeatedly executes a function at a set interval.
    • // Example
let intervalId = setInterval(() => {
    console.log("Interval executed!");
}, 1000); // Executes every second
clearInterval(intervalId); // Stops the interval
  • window.clearTimeout(): Cancels a timeout set by setTimeout().
  • window.clearInterval(): Cancels an interval set by setInterval().
Event Handlers
  • window.onload: Fired when the page and all of its resources have finished loading.
  • onresize: Executes when the window is resized.
    • // Example
window.onresize = () => {
    console.log("Window resized!");
};
  • onscroll: Executes when the window is scrolled.
    • // Example
window.onscroll = () => {
    console.log("Scrolling detected!");
};
  • onfocus and onblur: Detects focus and blur events.
    • // Example
window.onfocus = () => console.log("Window focused!");
window.onblur = () => console.log("Window blurred!");
  • onbeforeunload: Prompts before leaving the page.
    • // Example
window.onbeforeunload = () => {
    return "Are you sure you want to leave?";
};
Pop-ups
  • alert: Displays a message in an alert dialog.
  • window.confirm(): Displays a confirmation dialog box.
  • window.prompt(): Displays a dialog box with a text input.
Messaging
  • postMessage: Sends messages between windows.
    • // Example
// In the source window:
window.postMessage("Hello!", "*");

// In the target window:
window.onmessage = (event) => {
    console.log(event.data); // Logs "Hello!"
};
Window Relationships
  • window.frames: Returns a collection of all iframes in the current window.
  • parent: Refers to the parent window of an iframe.
    • // Example
console.log(window.parent); // Logs the parent window
  • top: Refers to the topmost window.
    • // Example
console.log(window.top); // Logs the topmost window
Storage
  • localStorage: Persistent storage in the browser.
    • // Example
localStorage.setItem("key", "value");
console.log(localStorage.getItem("key")); // Logs "value"
  • sessionStorage: Temporary storage cleared when the browser is closed.
    • // Example
sessionStorage.setItem("sessionKey", "sessionValue");
console.log(sessionStorage.getItem("sessionKey")); // Logs "sessionValue"
  • Cookies
    Setting Cookies 
    document.cookie = "username=JohnDoe; expires=Fri, 31 Dec 2024 23:59:59 UTC; path=/";

    Example: setCookie("username", "JohnDoe", 7)

    Reading Cookies 
    console.log(document.cookie);

    Example: getCookies() parses cookies into a JavaScript object.

    Deleting Cookies 
    document.cookie = "username=; expires=Thu, 01 Jan 1970 00:00:00 UTC; path=/";

    Example: deleteCookie("username") removes the cookie.

    Cookie Attributes
    • Path: Specifies the path where the cookie is accessible. 
      • document.cookie = "theme=dark; path=/admin";
    • Domain: Defines the domain where the cookie is accessible. 
      • document.cookie = "sessionId=12345; domain=example.com";
    • Secure: Sends cookies only over HTTPS.
      • document.cookie = "authToken=abcdef; secure";
    • HttpOnly: Prevents access to cookies via JavaScript (set on the server).
      • Set-Cookie: sessionToken=123456; HttpOnly
Browser Control

The window object provides several methods for controlling the browser window, including:

  • window.open(): Opens a new browser window or tab.
  • window.close(): Closes the current browser window.
  • window.resizeTo(): Resizes the browser window to a specified width and height.
DOM
  • window.document: Represents the DOM (Document Object Model) of the webpage.
Security Considerations

Cross-Origin Restrictions: The window object is subject to security restrictions, particularly in the context of cross-origin (CORS) requests. JavaScript in one window may not be able to access the properties of another window if they come from different origins. In this case the only way to communicate is using postMessage and onmessage methods
