Introduction to CSS

What is CSS?

CSS stands for Cascading Style Sheets. It’s a language used to style and design the layout of web pages. While HTML is used to create the structure of a webpage (like headings, paragraphs, images, etc.), CSS controls how those elements look on the page, including things like:

  • Colors (backgrounds, text, borders, etc.)
  • Fonts (font size, family, weight, etc.)
  • Spacing (margins, padding, line height, etc.)
  • Layout (positioning, alignment, display properties)
  • Animations and transitions (like hover effects or smooth scrolling)
How CSS works with HTML

CSS works by selecting HTML elements and applying styles to them. The process involves defining rules that specify how the content in HTML should be displayed. Here's how it works step-by-step:

  1. HTML Structure: HTML provides the content and structure of the web page (e.g., headings, paragraphs, images, etc.).
  2. CSS Selectors: In CSS, you define rules using selectors to target specific HTML elements. These selectors can be based on:
    • Element type: (e.g., p, h1)
    • Class: (e.g., .class-name)
    • ID: (e.g., #id-name)
  3. CSS Properties: After selecting an element, CSS defines the properties (such as color, font-size, margin, padding, background, etc.) that determine how the element should be displayed.
  4. Cascading: The "Cascading" part means that CSS rules can overlap. If multiple rules apply to the same element, the browser decides which one to apply based on:
    • Specificity: More specific selectors take precedence over general ones.
    • Source order: If two rules have the same specificity, the one that appears last in the CSS file will be applied.
  5. Rendering: The browser interprets the HTML and CSS, combining them to render the styled content on the screen. The result is the visually styled page that the user interacts with.
Different ways to apply CSS: Inline, Internal, External

There are three main ways to apply CSS to an HTML document:

  1. Inline CSS: Directly inside the HTML element using the style attribute. 
        <h1 style="color: red;">This is a red heading using Inline CSS</h1>
  2. Internal CSS: Within a <style> block in the head section of the HTML document. 
        <style>
        h1 {
        color: red; /* Applying the red color to the h1 element */
        }
    </style>
    <h1>This is a red heading using Internal CSS</h1>
  3. External CSS: Using a separate .css file linked to the HTML document. 
        <link rel="stylesheet" href="styles.css">
    <h1>This is a red heading using External CSS</h1>
            
        /* In styles.css */
    h1 {
        color: red; /* Applying the red color to the h1 element */
    }

CSS Syntax and Selectors

CSS syntax: selector { property: value; }

This is the basic structure of a CSS rule. It consists of three parts:

  • Selector: This part defines the HTML element you want to apply the style to. Example: p, div, or any class like .myClass.
  • Property: The property is the CSS attribute you want to change. For example, color, font-size, margin, etc.
  • Value: The value specifies what you want the property to be. For example, red, 16px, or auto.

Example: The rule p { color: red; } applies the color red to all paragraph elements (p) in the HTML document.

Types of Selectors
Simple Selectors
  • 1. Element Selector: Selects elements by their tag name.
    • p { color: red; }

    This will select all <p> elements and change their color to red.

  • 2. Class Selector: Selects elements with a specific class.
    • .myClass { color: blue; }

    This will select all elements with the class myClass and change their text color to blue.

  • 3. ID Selector: Selects an element with a specific ID.
    • #myId { font-size: 20px; }

    This will target the element with id="myId" and set its font size to 20px.

  • 4. Grouping Selectors: Allows you to apply the same style to multiple selectors by separating them with a comma.
    • h1, h2, p { color: purple; }

    This will apply the color: purple; style to <h1>, <h2>, and <p> elements. 

    .button, #submit { background-color: blue; }

    This will apply the background-color: blue; style to elements with the class button and the ID submit.

Combinator Selectors
  • 1. Child Selector: Selects direct children of an element.
    • div > p { color: green; }

    This will select all <p> elements that are direct children of a <div> element.

  • 2. Descendant Selector: Selects all descendants of an element. 
    • div p { color: orange; }

    This will select all <p> elements inside a <div>, even if they are deeply nested.

  • 3. Adjacent Sibling Selector: Selects the element immediately following a specific element.
    • h1 + p { color: purple; }

    This will select the first <p> element that directly follows an <h1> element.

  • 4. General Sibling Selector: Selects all sibling elements that follow a specific element.
    • h1 ~ p { color: pink; }

    This will select all <p> elements that are siblings of an <h1> element.

Pseudo-class Selectors
  • 1. :hover: Selects elements when the user hovers over them. 
    • a:hover { color: red; }

    This will change the color of a link to red when hovered over.

  • 2. :focus: Selects elements when they are focused (e.g., clicked or tabbed into).
    • input:focus { border: 2px solid blue; }

    This will add a blue border to the input field when it is focused.

  • 3. :nth-child(n): Selects elements based on their position in a parent.
    • li:nth-child(odd) { background-color: lightgray; }

    This will select all odd-numbered <li> elements and give them a light gray background.

  • 4. :first-child: Selects the first child element of a parent. 
    • p:first-child { font-weight: bold; }

    This will make the first <p> element inside its parent bold.

  • 5. :last-child: Selects the last child element of a parent. 
    • p:last-child { margin-bottom: 0; }

    This will remove the margin from the last <p> element inside a parent.

Pseudo-element Selectors
  • 1. ::before: Inserts content before an element's content.
    • p::before { content: "Note: "; font-weight: bold; }

    This will add the word "Note: " before each paragraph's text.

  • 2. ::after: Inserts content after an element's content.
    • p::after { content: "."; }

    This will add a period after each paragraph's text.

  • 3. ::first-letter: Targets the first letter of an element's content.
    • p::first-letter { font-size: 2em; color: red; }

    This will make the first letter of each paragraph larger and red.

  • 4. ::first-line: Targets the first line of text in an element. 
    • p::first-line { font-weight: bold; }

    This will make the first line of each paragraph bold.

Attribute Selectors
  • 1. Basic Attribute Selector: Selects elements that have a specific attribute.
    • a[href] { color: green; }

    This will select all <a> elements with the href attribute and make their text color green.

  • 2. Exact Attribute Value: Selects elements with a specific attribute value.
    • input[type="text"] { background-color: yellow; }

    This will select all <input> elements of type "text" and change their background color to yellow.

  • 3. Starts With: Selects elements whose attribute value starts with a given string.
    • a[href^="https://"] { color: blue; }

    This will select all links that start with "https://" and change their text color to blue.

  • 4. Ends With: Selects elements whose attribute value ends with a given string.
    • img[src$=".jpg"] { border: 1px solid black; }

    This will select all images with a `.jpg` file extension and add a black border.

  • 5. Contains: Selects elements whose attribute value contains a given substring.
    • a[href*="example"] { text-decoration: underline; }

    This will select all links whose href contains "example" and underline the text.

Basic Styling

Styling text: color, font-family, font-size, font-weight etc..

1. Color: The color property sets the color of the text.

p { color: red; }

This will set the color of the text inside all <p> elements to red.

2. Font-family: The font-family property defines which font should be used for the text.

p { font-family: Arial, sans-serif; }

This will apply the font "Arial" to all <p> elements. If "Arial" is unavailable, it will fall back to a generic sans-serif font.

3. Font-size: The font-size property controls the size of the text.

p { font-size: 16px; }

This will set the font size of the text in <p> elements to 16 pixels.

4. Font-weight: The font-weight property specifies the weight of the font (boldness).

p { font-weight: bold; }

This will make the text in all <p> elements bold.

5. Font-style: The font-style property specifies the style of the font, such as italic or normal.

p { font-style: italic; }

This will set the text in all <p> elements to italic.

6. Text-transform: The text-transform property controls the capitalization of the text.

p { text-transform: uppercase; }

This will make all text inside the <p> elements uppercase.

7. Letter-spacing: The letter-spacing property controls the space between characters.

p { letter-spacing: 2px; }

This will add 2 pixels of space between each character in the text inside <p> elements.

8. Line-height: The line-height property controls the amount of space between lines of text.

p { line-height: 1.5; }

This will set the line height of the text in <p> elements to 1.5 times the size of the font, giving more space between lines.

9. Text-align: The text-align property sets the alignment of the text inside an element.

p { text-align: center; }

This will center-align the text inside all <p> elements.

10. Text-decoration: The text-decoration property adds decoration to the text, like underline, line-through, or overline.

p { text-decoration: underline; }

This will underline the text inside all <p> elements.

11. Text-direction: The direction property sets the direction of text within an element. It is commonly used for handling text in languages that are written from right to left (like Arabic or Hebrew).

p { direction: rtl; }

This will set the text direction to right-to-left (rtl) for all <p> elements. By default, text flows left-to-right (ltr) in most languages.

12. Text-shadow: The text-shadow property adds shadow effects to text, making it stand out and adding depth to the text.

p { text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3); }

This will apply a shadow to the text inside all <p> elements, with a 2px horizontal and 2px vertical offset, a 4px blur radius, and a semi-transparent black color.

Styling background: background-color, background-image etc...

1. Background-color: The background-color property sets the background color of an element.

div { background-color: lightblue; }

This will set the background color of all <div> elements to light blue.

2. Background-image: The background-image property allows you to set an image as the background.

div { background-image: url('background.jpg'); }

This will set the background image of all <div> elements to the image located at 'background.jpg'.

3. Background-position: The background-position property defines the position of the background image.

div { background-position: center center; }

This will position the background image at the center of the <div> element.

You can also use values like top, bottom, left, right, or percentages for precise positioning.

4. Background-size: The background-size property specifies the size of the background image.

div { background-size: cover; }

This will make the background image cover the entire <div> element while maintaining its aspect ratio.

Other values include contain to fit the image inside the element without cropping it.

5. Background-repeat: The background-repeat property defines whether and how the background image repeats.

div { background-repeat: no-repeat; }

This will prevent the background image from repeating inside the <div> element.

Other values include repeat (default), repeat-x (repeats horizontally), and repeat-y (repeats vertically).

6. Background-attachment: The background-attachment property controls whether the background image scrolls with the page or stays fixed in place.

div { background-attachment: fixed; }

This will keep the background image fixed in place when scrolling the page.

Other values include scroll (default, the image scrolls with the content) and local (the image scrolls with the content inside the element).

7. Background (Shorthand): The background property is a shorthand that allows you to set multiple background properties in one line.

div { background: lightblue url('background.jpg') no-repeat center center / cover fixed; }

This combines the following properties:

  • background-color: lightblue
  • background-image: url('background.jpg')
  • background-repeat: no-repeat
  • background-position: center center
  • background-size: cover
  • background-attachment: fixed

Box Model and Layout

Box Model: Understanding how padding, margin, and borders affect layout

The Box Model in CSS refers to the structure of an element and how content, padding, border, and margin work together:

The Box Model is crucial for understanding how elements occupy space on a webpage. It consists of the following parts:

  • Content
  • Padding
  • Border
  • Margin

Box Model Layout

The image above shows how the padding, border, and margin affect an element's total size and layout.

1. Content: The actual content of the element, where text or other content appears.

div { width: 200px; height: 100px; }

The content area can have defined width and height, and it is where the content of the element, like text or images, is placed.

2. Padding: Padding is the space between the content and the border of the element. It is used to create space inside the element.

div { padding: 20px; }

This will create a 20-pixel space inside the <div> element between its content and the border.

3. Border: The border wraps around the content and padding. You can set its thickness, style, and color.

div { border: 2px solid black; }

This will add a 2-pixel black border around the <div> element.

4. Margin: Margin is the space outside the element, between the element's border and other elements.

div { margin: 10px; }

This will create a 10-pixel space outside the <div> element.

Create a basic webpage that uses HTML and CSS. The page will include a heading, a paragraph, a list, and an image, styled using inline, internal, and external CSS, along with various CSS selectors.

Requirements:

  • HTML Structure:

    • Use the universal selector to remove margins/paddings.

Recipe Page Design with HTML and CSS

Objective: Create a visually appealing recipe page for a fictional dish. Use HTML and CSS to design the page with multiple sections such as the recipe name, ingredients, preparation steps, and an image of the dish. You will use background colors, text styling, and images to enhance the visual appeal of the page.

Requirements:

  • HTML Structure: Create a page that includes the following sections:
    • Recipe Name: A heading that includes the name of the recipe (e.g., "Delicious Chocolate Cake").
    • Ingredients: A list of ingredients required for the recipe (use an unordered list <ul>).
    • Preparation Steps: Numbered steps to explain how to make the recipe (use an ordered list <ol>).
    • Image: An image of the dish you are showcasing. Include an image and provide a caption below it (use <img> and <p> for the caption).
  • CSS Styling: Apply the following styles using internal CSS:
    • Background Colors: Set different background colors for each section (e.g., Recipe Name, Ingredients, Preparation Steps, Image Section).
    • Text Styling:
      • Use large, bold font for the Recipe Name.
      • Style the ingredients and steps with different font sizes and line heights for readability.
      • Make the Preparation Steps list numbered (<ol>).
    • Image: Style the image to be responsive so it fits well on different screen sizes. Add a caption below the image in italics.

Section Breakdown:

  • Recipe Name: Use a bold background color (e.g., red or orange) and display the recipe name in a large, bold font.
  • Ingredients: Use a soft background color (e.g., light green) and list the ingredients with bullet points.
  • Preparation Steps: Use a light background color (e.g., yellow or beige) and display the preparation steps in a numbered list.
  • Image Section: Use a clean background color (e.g., light gray or white) to showcase the image. Add a caption in italics under the image.

Additional Notes:

  • Ensure the page is mobile-friendly and looks good on different screen sizes (use responsive design techniques like percentage-based widths for images and text containers).
  • Feel free to be creative with your design, but make sure the page is organized and easy to read.
  • Use CSS selectors for styling specific elements (like background color, text styling, and image handling). Try to use element, class, and ID selectors where appropriate.

Evaluation Criteria:

  • Proper structure of HTML with all required sections.
  • Correct application of background colors and text styling.
  • Responsive design for the image and overall page.
  • Use of appropriate CSS selectors for styling the page.
