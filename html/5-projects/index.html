HTML Projects

Assignments

  1. Project: Create a personal homepage using only semantic HTML. Include sections for About Me, Education, Skills, and Contact Form. Use <header>, <section>, <footer>, <form>, and semantic input types like email and date.

  2. Project: Create a product landing page using only HTML. Include:
    • A navigation menu using <nav> with internal anchor links (e.g., #features, #contact).
    • A header section with the product name and tagline using <header> and <h1>.
    • A product description section using semantic tags like <section>.
    • An embedded product video using <iframe> or <video>.
    • A table listing different product plans/features using <table>, <thead>, <tbody>, and <caption>.
    • Use <footer> to include copyright and contact details.
